Rotating Circular CTA Animation
📌 Overview

This project demonstrates a modern circular text animation used as a call-to-action (CTA).
The text rotates smoothly around a central arrow, creating an engaging and interactive UI component.

🛠️ Technologies Used
HTML5
CSS3 (Animations & Transforms)
SVG (Text Path for circular text)
✨ Features
Infinite smooth rotation animation
Circular text using SVG path
Center arrow with hover interaction
Lightweight and responsive design
🎯 Use Cases
Portfolio websites
Creative agency landing pages
“Latest Work” or “Explore” sections
Interactive UI components
🚀 How to Use
Copy the HTML structure into your project
Add the CSS styles
Customize text and animation speed as needed
